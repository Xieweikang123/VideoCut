# VideoCut - 简单视频剪辑软件规格文档

## 1. 项目概述

**项目名称:** VideoCut
**项目类型:** 桌面应用 (Tauri + React)
**核心功能:** 简单的视频剪辑工具，支持视频的裁剪、分割和合并
**目标用户:** 需要进行基础视频编辑的用户

## 2. 技术栈

| 层级 | 技术 |
|------|------|
| 后端框架 | Tauri 2.x (Rust) |
| 前端框架 | React 18 + TypeScript |
| 构建工具 | Vite 6 |
| 状态管理 | Zustand 5 |
| 样式 | TailwindCSS 3 |
| 视频处理 | FFmpeg (通过 Rust 命令封装) |

## 4. 功能规格

### 4.1 视频导入
- 支持格式: MP4, AVI, MOV, MKV, WebM, FLV, WMV
- 通过系统文件对话框选择
- 导入后自动添加到时间线
- 获取视频时长和分辨率信息

### 4.2 视频预览
- 使用 HTML5 video 元素
- 支持播放/暂停控制
- 显示当前播放时间和总时长
- 播放头与时间线同步

### 4.3 时间线编辑
- 显示所有视频片段
- 点击时间线定位播放头
- 点击片段选中
- 拖拽排序片段

### 4.4 裁剪 (Trim)
- 设置 in point (入点)
- 设置 out point (出点)
- 通过拖拽片段边缘调整
- 实时预览裁剪效果

### 4.5 分割 (Split)
- 在播放头位置分割片段
- 将一个片段分成两个
- 保留分割点前后的内容

### 4.6 删除
- 删除选中的片段
- 从时间线移除

### 4.7 导出
- 导出合并后的完整视频
- 支持选择输出路径
- 使用 FFmpeg 进行实际处理

## 5. 数据结构

### 5.1 Clip (视频片段)
```typescript
interface Clip {
  id: string;           // 唯一标识
  sourcePath: string;  // 源文件路径
  startTime: number;    // 在时间线上的开始时间
  endTime: number;      // 在时间线上的结束时间
  inPoint: number;      // 入点 (源文件内)
  outPoint: number;     // 出点 (源文件内)
}
```

### 5.2 Project (项目)
```typescript
interface Project {
  clips: Clip[];         // 所有片段
  playheadTime: number; // 当前播放头位置
  zoom: number;          // 时间线缩放比例
}
```

## 6. FFmpeg 集成

### 6.1 裁剪命令
```bash
ffmpeg -i input.mp4 -ss 00:00:10 -to 00:00:30 -c copy output.mp4
```

### 6.2 合并命令
```bash
ffmpeg -f concat -safe 0 -i list.txt -c copy output.mp4
```
